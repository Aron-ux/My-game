# Security Policy

This is a local Godot prototype and currently has no public security support window.

## Sensitive areas

- Save files under `user://`
- Future Steam/GodotSteam integration
- Local MCP tooling under `addons/godot_mcp/`
- Any future networked or platform-authenticated feature

## Reporting

For now, report issues directly to the project owner/maintainer. Do not publish exploit details for unreleased builds.

## Current limitations

- No production backend exists.
- No multiplayer or account system exists.
- Steam integration is documented as a future adapter only; it is not active by default.
