# UI、菜单、存档与音频

## 主菜单

主菜单场景：
`res://scenes/main_menu.tscn`

当前主菜单功能：

- 继续游戏
- 主线模式
- 挑战模式（当前禁用）
- 无尽模式（当前禁用）
- 开发者模式
- 设置
- 退出

## 主菜单视觉规则

- 当前首屏即主菜单
- 背景图需要随窗口和全屏自适应拉伸
- 当前主菜单背景资源来自 `assets/demo.jpg`

## 设置菜单

主菜单和暂停菜单都支持：

- 背景音乐音量滑条
- 背景音乐静音开关

## HUD

当前 HUD 方向：

- 常驻显示：时间、等级
- 核心显示：经验条、HP、MP
- Boss 出现时顶部切换为 Boss 名和 Boss 血条
- 高级信息可通过按键切换显示

## 暂停菜单

触发方式：`ESC`

当前暂停菜单功能：

- 继续游戏
- 重新开始
- 返回主菜单
- 音量调节
- 静音开关

## 存档结构

核心脚本：
`scripts/save_manager.gd`

存档根目录：
`user://story_slots`

当前存档结构：

- `meta.json`
  记录最近使用的存档位等元信息

- `slot_x/story_profile.json`
  主线存档资料

- `slot_x/run_save.json`
  战斗中的继续存档

- `slot_x/run_save_backup.json`
  战斗存档备份

## 当前存档位规则

- 当前固定 `3` 个故事存档位
- 每个存档位相互独立
- 继续游戏默认基于最近使用的存档位

## 继续游戏规则

- 强退或中途返回主菜单后，允许继续当前战斗
- 继续目标通常回到 `main.tscn`
- 如果没有战斗存档但有主线存档，则回到主线准备流程

## 主线准备界面

场景：
`res://scenes/story_prep.tscn`

当前功能定位：

- 作为“轻营地 / 关卡前准备”的功能版入口
- 提供队伍顺序、角色风格切换、进入下一关

当前角色池数据定义：

- 已开放：
  剑士、枪手、术师
- 预留位：
  角色4、角色5

## 故事档内容

当前默认故事档包含：

- 当前章节 / 当前关卡索引
- Boss 核心碎片
- 已解锁角色
- 当前队伍顺序
- 角色已解锁风格
- 角色已装备风格

## 音频系统

核心脚本：
`scripts/bgm_player.gd`

当前能力：

- 读取和保存音量
- 静音开关
- 循环播放
- 暂停与恢复
- 记录上次播放位置并尝试续播

## 当前 BGM 规则

- 主菜单与游戏内使用不同 BGM
- 暂停时可暂停游戏 BGM
- 继续游戏时可尝试从上次播放进度附近恢复
